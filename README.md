# AI Work Tracker by Aleksandar Kanchev

AI Work Tracker is a desktop app that works on Mac and Windows that tracks work activity and provides efficiency insights.

## Features
- Automatically tracks screen activity and categorizes it as productive or non-productive.
- Generates efficiency scores based on time spent working.
- Provides a daily summary of activities.
- Works entirely in the background.
